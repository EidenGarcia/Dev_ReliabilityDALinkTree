# Reliability DA · LinkTree

Internal link hub for the LUMA Energy Reliability DA team — quick access to guidelines, Microsoft Forms, trackers/reports, and interactive engineering tools.

## Structure

```
├── index.html                                  # Home (LinkTree)
├── guides5.html                                # Guideline Library
├── da_forms.html                               # Microsoft Forms
├── da_trackers.html                            # Trackers & Reports
├── tools.html                                  # Tools hub
├── CMI_Avoidance_Metric.html                   # Power BI embed
├── ip_calculator.html                          # IP Calculator
├── Metering Calculator.html                    # Metering Check
├── qr_code_generator_tool.html                 # QR Generator
├── ansi_overcurrent_calculator_with_graph.html # ANSI Overcurrent Calculator
├── S&C_K_Speed FuseLinks_TCCalculator.html     # TCC Coordination Calculator
├── styles.css                                  # Shared LUMA-branded stylesheet
├── assets/images/                              # Logos & guide thumbnails
└── docs/                                       # Reference PDFs
```

HTML pages intentionally remain at the repo root so existing bookmarks, QR codes, and shared URLs keep working.

## Branding

Styled to the LUMA Energy palette (lumapr.com):

| Token | Hex |
|---|---|
| Navy | `#14214B` |
| Green | `#79A941` |
| Blue accent | `#1F3A7A` |
| Background | `#EEF2F6` / white cards |

All pages share `styles.css`; the palette lives in the `:root` CSS variables, so future rebrands only require editing that block.
